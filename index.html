Expenditrack


The core functionality for Expenditrack will consist of an Android Application that will allow the user to manage their receipts/invoices in a “modern” way. By co-operation between the application and Google Vision API the user will have the ability to use OCR(Optical Character Recognition) in order to make use of the textual content of an image taken on their device, the text extracted by OCR is formatted to JSON and passed back to a RESTful web service (hosted on Microsoft Azure) in order to filter the important information which is then stored in a Database. This will allow the tracking of important expenditure to the user.



Library Rental System


The System I have developed is a Library Rental System, it can carry out all the functions of a normal Library System, having the ability to manage customers, books, users and generate reports. There are two user types Librarian and Admin. The system will allow all users’ to create/rent/return a book and add/delete/modify a customer account. An Admin has access to add/delete/modify a user account and full access to Reports for both Admin and Librarian. Reports are generated as PDF. The reports consist of a list of how many times each book has been rented in order to easily spot the most popular, the Rentals and Returns per person are also available.



Garage Booking System


The system allows any user that has access to create a booking, storing all of the relevant customer details and information regarding their car. Bookings are made and a time and date is decided by the user and the customer suiting the needs of both. The system then checks for availability once a booking is being created. If a deposit is needed to create the booking the system will notify the user. The system provides a timetable to the user to allow them to select the time and date desired. Once availability is confirmed and a deposit is made if necessary then the system prompts the user to enter the customer’s full details for review upon entry.
